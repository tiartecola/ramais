# Ferramenta de Ramais — Artecola

Sistema interno desenvolvido para facilitar a consulta, organização e gerenciamento dos ramais corporativos da Artecola.

A ferramenta tem como objetivo centralizar as informações de contato interno da empresa, tornando mais rápido o acesso aos ramais por setor, unidade, colaborador ou área responsável.

---

## 📌 Sobre o projeto

A ferramenta de Ramais da Artecola foi criada para otimizar a comunicação interna entre colaboradores, unidades e setores da empresa.

Com ela, é possível consultar ramais de forma simples, rápida e organizada, reduzindo a dependência de planilhas manuais ou listas descentralizadas.

---

## 🚀 Funcionalidades

- Consulta de ramais internos
- Busca por nome, setor, unidade ou departamento
- Organização dos contatos por áreas da empresa
- Interface simples e intuitiva
- Centralização das informações de telefonia interna
- Facilidade na atualização dos dados
- Apoio à comunicação corporativa
